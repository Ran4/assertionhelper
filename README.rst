

Installation
------------

Local installation:

::

    git clone github.com/ran4/assertionhelper
    cd assertionhelper
    pip install .
    #pip install -e . #installs locally with symlink
    
    
Generating docs
---------------

::

    sphinx-apidoc -F -o docs assertionhelper
    cd docs
    make html
